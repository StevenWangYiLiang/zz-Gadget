# 金融小工具集（zz-Gadget）设计文档

- 日期：2026-07-15
- 状态：已确认，待进入实现计划
- 平台：网页版 / PWA（安卓手机浏览器打开即用，可添加到主屏幕）

## 1. 目标与范围

一个自包含的网页小工具集，面向移动端（安卓）使用，当前聚焦金融计算器。
本期交付两个工具，并搭建可持续扩展的「工具集」骨架，后续新增工具照模板填空即可。

- 工具1：免息价值折现器
- 工具2：CAGR 计算器

不做（YAGNI）：账号系统、云同步、后端、App 上架、多语言。

## 2. 架构与文件结构

单个自包含 `index.html`，HTML / CSS / JS 全部内联，无外部依赖、不联网、无构建步骤。

```
zz-Gadget/
├─ index.html                 # 全部内容：首页菜单 + 工具页 + 计算逻辑 + 样式
├─ README.md                  # 使用说明 + 如何新增工具
└─ docs/superpowers/specs/    # 本设计文档
```

- **路由**：哈希路由。`#/` 首页；`#/discount` 折现器；`#/cagr` CAGR。切换不刷新、不联网。
- **工具注册表**：一个 `TOOLS` 数组，每项 `{ id, name, desc, render }`。首页据此自动生成卡片列表；路由据 `id` 派发到对应 `render`。
- **纯函数分离**：计算逻辑写成不碰 DOM 的纯函数（`pvAnnuity(...)`、`cagr(...)`），便于单独测试与复用。
- **可离线**：直接本地打开 `index.html` 即可运行；无网络请求。

## 3. 计算规格

### 工具1 免息价值折现器

前提：免息分期意味着「现金价」= 各期还款之和，即直接支出总价 = 月供 × 期数。
分期后每笔在未来支付，按折现率贴现到今天，节省的就是这笔钱的时间价值。

- 输入
  - 月供 `M`（元，> 0）
  - 期数 `n`（正整数，> 0）
  - 年化折现率 `r`（默认 3%，用户可改，允许 0）
- 口径
  - 月折现率 `i = r / 12`（名义年化 / 12，例：3% → 0.25%）
  - 普通年金（ordinary annuity）：每期期末支付，首期在第 1 个月末
- 公式
  - 现值 `PV = M × [1 − (1 + i)^(−n)] / i`；当 `i = 0` 时退化为 `PV = M × n`
  - 直接支出总价 `Total = M × n`
  - **节省的时间价值 `Saving = Total − PV`**
  - 节省比例 `SavingRate = Saving / Total`
- 输出
  - 主结果（大字）：节省时间价值 `Saving`
  - 辅助（小字）：现值 `PV`、直接支出总价 `Total`、节省比例 `SavingRate%`、所用月率 `i`
- 边界：`M ≤ 0` 或 `n ≤ 0` → 温和提示，不计算；`r = 0` → `Saving = 0`。

### 工具2 CAGR 计算器

- 输入
  - 期初金额 `V0`（> 0）
  - 期末金额 `V1`
  - 年数 `N`（> 0，允许小数）
- 公式
  - `CAGR = (V1 / V0)^(1/N) − 1`
  - 总增长倍数 `V1 / V0`；总涨幅 `V1 / V0 − 1`
- 输出
  - 主结果（大字）：`CAGR = X%`
  - 辅助（小字）：总增长倍数、总涨幅%、年数
- 边界：`V0 ≤ 0` 或 `N ≤ 0` → 提示无法计算；`V1 < 0` → 提示（负终值 CAGR 无意义）；`V1 < V0` → 正常给出负增长率。

## 4. 界面与交互

- **风格：扁平简洁**。移动端优先、单列布局、留白充足；单一强调色；系统字体；不用重阴影 / 渐变 / 拟物。
- 输入框大、`inputmode="decimal"` 调起数字键盘；大按钮易点。
- **即时计算**：输入变化即时更新结果，无需点按钮。
- 结果区：主结果大字突出，下面小字列出中间量。
- 输入非法：结果区红字温和提示，不弹窗。
- 顶部提供返回首页入口。
- 金额显示：千分位分隔、保留 2 位小数；比例 / 增长率显示为百分比（保留 2 位小数）。

  - `V0 = 100, V1 = 100, N = 5` → `0%`

## 6. 扩展模板（新增工具三步）

1. 写纯计算函数（不碰 DOM）。
2. 写 `render` 函数：渲染表单 + 绑定即时计算 + 输出结果。
3. 在 `TOOLS` 注册 `{ id, name, desc, render }`。

首页卡片与路由自动接入，无需改动其它代码。README 记录这套模板。
